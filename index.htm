LINQ中的哲学与薛定谔的猫

今天想到一个有意思的东西。
List<int> a = new List<int>();
Console.WriteLine(a.All(n => n > 10));//true
Console.WriteLine(a.All(n => n < 10));//true
然而，众所周知的是，bool的默认值是false的。
看来是作者故意想告诉我们：未来还未确定，all() == true。
